Disqus是什么

Disqus（/dɪsˈkʌs/，与英语“discuss”同音）是一家使用社交网络形式，向网络社区提供网站留言服务的公司。该公司的平台提供不同的功能，例如与不同社交网络服务链接、社交网络、用户个人文件、垃圾宣传及审核工具、数据分析、电邮通知和在移动设备留言等。它是于2007年由Daniel Ha和Jason Yan成立为一家Y Combinator初创公司。

安装

首先安装插件，到GitHub网页跟着指示安装。

bbob add -a https://github.com/Reknij/bbob-plugin-disqus/releases/download/v1.0.0/bbob-plugin-disqus.rar

这里我安装的是v1.0.0版本。

配置

在./configs/目录下新建一个配置文件bbob-plugin-disqus-config.json，接着输入以下内容

{
    "shortName": "example"
}

请将shortName改成你自己的。关于shortName请自行到Disqus注册，可以得到你的shortName，一个Disqus账号可以有多个shortName，一个shortName代表一个网站。

使用

输入命令后：

bbob g -p

打开网页浏览localhost，打开其中一个文章即可看到效果。如果更改过预览端口，请自行通过指定的端口查看预览网页。
