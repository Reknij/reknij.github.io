第一次使用Hexo blog

2021年03月22日 209 字 大概 1 分钟

我建立博客的缘故

网络上看见很多博客，感觉很高大尚，很早就想试试了。为此，我学习许多前后端的知识，虽然只是基础，不过也很满意了，未来我也想尝试建立自己的CMS网站。创建此文章时我的基础有HTML + CSS + JS, Vue, ExpressJS。

开始搭建博客

我使用GitHub Pages + Hexo来搭建博客。教程网上找的。根据此文章学习搭建。

使用主题

我使用的主题为next, 在主目录下npm install hexo-theme-next安装主题，并修改_config.yml中的theme: default_theme为theme: next即可。将指定主题中的_config.yml复制一份到主目录并命名为_config.next.yml即可。

新建文章

新建文章为hexo new post_name。也可新建页面hexo new page page_name。
